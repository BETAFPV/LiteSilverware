Precompiled binaries for the bwhoop quad

bwhoop_devo720_ch6.hex - intended for devo / module , ch6 ( devo ) switches acro / level, rate 720

b03-pro_factory_fw.bin - factory firmware for the pro, will also remove alt hold from factory b03
